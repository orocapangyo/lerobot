# LeRobot 프로젝트 소개

## 개요

LeRobot은 **실제 로봇을 위한 모방 학습(Imitation Learning)과 강화 학습(Reinforcement Learning)**을 위한 오픈소스 프레임워크입니다. HuggingFace에서 개발하고 있으며, 최신 로봇 학습 알고리즘을 누구나 쉽게 사용할 수 있도록 설계되었습니다.

## 주요 특징

### 🎯 최첨단 알고리즘
- **Imitation Learning**: ACT, Diffusion Policy, VQ-BeT 등
- **Reinforcement Learning**: SAC, TDMPC, HIL-SERL 등
- **Vision-Language-Action 모델**: SmolVLA, Pi0, Pi0.5, GROOT 등

### 📊 풍부한 데이터셋
- **150개 이상**의 공개 데이터셋
- HuggingFace Hub 통합으로 간편한 다운로드
- LeRobotDataset v3.0 표준 포맷

### 🤖 다양한 로봇 지원
- **저가형 로봇**: SO-100, SO-101 (~$200-400)
- **연구용 로봇**: Koch, ALOHA
- **고급 로봇**: HopeJR, Reachy2, LeKiwi
- **커스텀 로봇** 통합 가능

### 🎮 시뮬레이션 환경
- **ALOHA**: 양팔 조작 작업
- **PushT**: 2D 푸싱 작업
- **Libero**: 가정 작업 벤치마크 (10종)
- **MetaWorld**: 조작 작업 벤치마크 (50종)

### 🔧 완벽한 툴체인
- 원격 조작(Teleoperation) 시스템
- 데이터 수집 및 시각화
- 모델 훈련 및 평가
- 실시간 추론 (동기/비동기)

## 프로젝트 구조

```
lerobot/
├── src/lerobot/
│   ├── datasets/          # 데이터셋 로딩 및 처리
│   ├── policies/          # 정책 구현 (ACT, Diffusion 등)
│   ├── envs/              # 시뮬레이션 환경
│   ├── processor/         # 데이터 전처리/후처리
│   ├── robots/            # 로봇 하드웨어 통합
│   ├── cameras/           # 카메라 인터페이스
│   ├── motors/            # 모터 제어
│   ├── teleoperators/     # 원격 조작
│   ├── rl/                # 강화학습 컴포넌트
│   └── scripts/           # CLI 도구
├── examples/              # 예제 코드
├── docs/                  # 문서
└── tests/                 # 테스트
```

## 사용 사례

### 1. 시뮬레이션에서 학습
```python
# PushT 환경에서 Diffusion Policy 훈련
python -m lerobot.scripts.lerobot_train \
  --policy-name diffusion \
  --dataset-repo-id lerobot/pusht \
  --env-name PushT-v0
```

### 2. 사전 훈련 모델 사용
```python
from lerobot.policies.act import ACTPolicy

# HuggingFace Hub에서 모델 로드
policy = ACTPolicy.from_pretrained("lerobot/act_aloha_insertion")

# 추론 실행
action = policy.select_action(observation)
```

### 3. 실제 로봇으로 데이터 수집
```bash
# 원격 조작으로 데모 수집
python -m lerobot.scripts.lerobot_record \
  --robot-path lerobot/robots/configs/so100.yaml \
  --fps 30 \
  --num-episodes 50
```

## 지원되는 정책

| 정책 | 타입 | 주요 환경 |
|------|------|----------|
| **ACT** | Imitation Learning | ALOHA |
| **Diffusion** | Imitation Learning | PushT, ALOHA |
| **VQ-BeT** | Imitation Learning | 범용 |
| **TDMPC** | Model-based RL | 범용 |
| **SmolVLA** | Vision-Language-Action | 범용 |
| **Pi0/Pi0.5** | Flow-matching VLA | ALOHA |
| **GROOT** | VLA | 범용 |
| **SAC** | Model-free RL | 범용 |
| **RTC** | 추론 최적화 | Pi0, SmolVLA |

## 지원되는 로봇

### 저가형 ($200-400)
- **SO-100**: 5-DoF 로봇 팔
- **SO-101**: SO-100 개선 버전

### 연구용
- **Koch v1.1**: 모듈형 6-DoF 로봇 팔
- **ALOHA**: 양팔 로봇 시스템
- **Bi-SO-100**: 양팔 SO-100

### 고급
- **HopeJR**: 휴머노이드 로봇 팔
- **Reachy2**: 휴머노이드 로봇
- **LeKiwi**: 모바일 매니퓰레이터

## 주요 논문

LeRobot에 구현된 알고리즘의 원문 논문:

### Imitation Learning
- **ACT**: "Learning Fine-Grained Bimanual Manipulation with Low-Cost Hardware" (ALOHA 논문)
- **Diffusion Policy**: "Diffusion Policy: Visuomotor Policy Learning via Action Diffusion"
- **VQ-BeT**: "Behavior Generation with Latent Actions"

### Vision-Language-Action
- **SmolVLA**: "Small Vision-Language-Action Models for Robotic Manipulation"
- **Pi0**: "Flow Matching for Generalist Policies"
- **GROOT**: (참조 논문 확인 필요)

### Reinforcement Learning
- **SAC**: "Soft Actor-Critic: Off-Policy Maximum Entropy Deep Reinforcement Learning"
- **TDMPC**: "Temporal Difference Learning for Model Predictive Control"
- **HIL-SERL**: "Human-in-the-Loop Imitation Learning using Remote Teleoperation"

### 추론 최적화
- **RTC**: "Real-Time Chunking for Flow-Matching Policies"
